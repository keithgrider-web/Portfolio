Maintaining HEPA-Safe Airflow Under Filter Loading via Differential Pressure Control
Challenge
Market-leading products were tuned so the fan wouldn’t exceed HEPA filter max flow when
new—but performance degraded as filters loaded with dust. Our team designed a system
that maintains maximum airflow while staying within filter limits.
contribution
Architecture development, filter selection, circuit + PCB, and control algorithm.
Process:
Air Scrubber
Product Performance Improvement: Developed a system using a high flow
rate fan and a control system to ensure optimal airflow, even as filters load
with dust.
Cost-Effective Control System: Implemented a split phase AC motor control
with non-contact speed measurement. Simpler to use BLDC, but more cost
effective to use A/C induction.
Fan and Filter Optimization: The high flow rate fan with the differential
pressure measurement algorithm allows regulation of flow rate to maintain
optimal airflow. This addresses the performance degradation of existing
products as filters load.
