Development

Current market leading products use a fan 
which does not exceed the HEPA flow rate of 
the filter when new but suffer degraded 
performance as the filters load with dust

To improve performance, we developed a 
system which uses a high flow rate fan and a 
control system to ensure the flow rate was 
did not exceed the filter maximum

To do this and meet cost targets, we selected 
a split phase AC motor control, developed a 
non-contact speed control system and 
developed an algorithm utilizing differential 
pressure measurement to control flow rate
